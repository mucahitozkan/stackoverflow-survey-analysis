# Stack Overflow Developer Survey 2025 Analysis

## Overview
This project analyzes the Stack Overflow Developer Survey 2025 
to uncover trends in the global developer community, including 
programming language popularity, salary trends, and remote work patterns.

## Key Questions
- Which countries have the most developer survey participants?
- What are the most popular programming languages?
- How does experience affect salary?
- How common is remote work among developers?
- Does education level impact salary?

## Key Findings
- Most respondents are from the US, Germany, and India
- JavaScript, HTML/CSS, SQL, and Python are the most used languages
- Salary increases dramatically after the first 3 years of experience
- Over 65% of developers work fully remote or hybrid
- Higher education correlates with higher salary

## Tools Used
- Python, Pandas, Matplotlib, Seaborn
- SQL (SQLite)
- Jupyter Notebook
- Git & GitHub
